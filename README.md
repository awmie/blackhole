# Blackhole

A real-time 3D blackhole simulation built with Three.js and Next.js.

![Demo](./blackhole.gif)

## Features

- Gravitational Lensing Shader
- Accretion Disk Visualization
- Orbital Physics Simulation
- Collision Detection
- Interactive Controls

## Getting Started

```bash
git clone https://github.com/awmie/blackhole.git
cd blackhole
pnpm install
pnpm dev
```

Open http://localhost:9002

## Tech Stack

Next.js 路 Three.js 路 TypeScript 路 Tailwind CSS 路 Radix UI
