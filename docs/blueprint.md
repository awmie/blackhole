# **App Name**: Event Horizon Visualizer

## Core Features:

- Appearance customization: Configure the appearance of the black hole and its accretion disk via UI sliders.
- Real-time rendering: Simulate the gravitational lensing effect of a black hole using Three.js.
- Simulation data: Display relevant data about the simulation (e.g. Schwarzschild radius, camera position).

## Style Guidelines:

- Primary color: Dark purple (#7851A9), evoking the vastness and mystery of space.
- Background color: Near-black (#08050F), setting a dark, immersive scene.
- Accent color: Bright cyan (#70DBFF), to highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, technical feel.
- Use simple, line-based icons to represent different simulation parameters.
- Divide the screen into a simulation view and a control panel for adjusting settings.
- Use subtle animations to transition between different simulation states and settings.
